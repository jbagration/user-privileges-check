# Тестовое задание

## Описание

Необходимо разработать веб-приложение для проверки привилегий пользователя.

## Требования

1. Пользователь вводит свое имя в форму на веб-странице.
2. При отправке формы сервер проверяет имя пользователя и возвращает его привилегии.
3. Если пользователь имеет привилегии, отображается сообщение с указанием привилегии.
4. Если пользователь не найден, отображается сообщение о том, что пользователь не найден.

## Структура проекта

- `index.html` - Главная страница с формой для ввода имени пользователя.
- `result.html` - Страница с результатом проверки привилегий пользователя.
- `index.js` - Серверный файл для обработки запросов и проверки привилегий пользователя.
- `public/` - Каталог для статических файлов (HTML, CSS, JavaScript).
- `server.bat` - Скрипт для запуска сервера.

## Инструкции по запуску

1. Убедитесь, что у вас установлен [Node.js](https://nodejs.org/).
2. Склонируйте репозиторий и перейдите в корневую директорию проекта.
3. Установите зависимости командой `npm install`.
4. Запустите сервер командой `server.bat`.
5. Откройте браузер и перейдите по адресу `http://localhost:3000`.

## Пример использования

1. Откройте `index.html` и введите имя пользователя.
2. Нажмите кнопку "Проверить".
3. На странице `result.html` отобразится результат проверки привилегий.

## Замечания

- Имя пользователя `admin` имеет привилегию `администратор`.
- Имя пользователя `user` имеет привилегию `пользователь`.
- Все остальные имена считаются несуществующими.
